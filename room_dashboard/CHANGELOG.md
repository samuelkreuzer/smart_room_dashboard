# Changelog

All notable changes to this project will be documented in this file.

This project follows a simple versioning model suitable for a small local application.

## [1.0.0] - 2026-06-13

### Added

- Initial Smart Room Dashboard release.
- Fullscreen responsive dashboard UI.
- Python standard-library web server.
- Weather integration through Open-Meteo.
- Google Calendar support through private iCal URLs.
- Local JSON to-do list.
- Music status through `playerctl`.
- Network status module.
- RSS and Atom feed reader.
- Raspberry Pi systemd service template.
- Chromium kiosk autostart template.
- Raspberry Pi installer script.
- Public documentation, contribution guide, security policy, and MIT license.
